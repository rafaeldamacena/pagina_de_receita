# Página de Receita - Bolo de Rolo

## Projeto desenvolvido durante a trilha da Rocketseat com foco nos fundamentos de HTML e CSS.
A proposta original da aula era criar uma página de receita (bolo de café), mas adaptei o projeto para representar um clássico da culinária nordestina: o bolo de rolo.

### Tecnologias utilizadas:
HTML5
CSS3
Google Fonts

### O que aprendi:
Neste projeto, pratiquei conceitos fundamentais como:
Estruturação semântica com HTML (main, section, h1, h2)
Organização de conteúdo em listas (ul e ol)
Estilização com CSS
Uso de box-sizing para controle de layout
Centralização de conteúdo com margin: auto
Aplicação de fontes externas com Google Fonts
Uso de imagens de fundo (background-image)

### Funcionalidades:
Exibição de uma receita completa
Separação por seções:
Sobre a receita
Ingredientes
Modo de preparo
Layout centralizado e estilizado
Imagem ilustrativa do prato
